# AI English Coach Application

Time-Boxed AI English Coach web application with voice interaction.

## 🚀 起動方法 (How to Start)

1. **ターミナルを開く**
   VS Codeのメニューから `Terminal` -> `New Terminal` を選択、またはコマンドプロンプトを開きます。

2. **プロジェクトフォルダに移動**
   ```bash
   cd C:\Users\vamos\.gemini\antigravity\scratch\ai-english-coach
   ```
   *(すでにこのフォルダにいる場合はスキップ)*

3. **依存パッケージのインストール** (初回のみ)
   ```bash
   npm install
   ```

4. **アプリの起動**
   ```bash
   npm run dev
   ```

5. **ブラウザでアクセス**
   ターミナルに表示されるURL（通常は `http://localhost:5173`）を `Ctrl + クリック` するか、ブラウザのアドレスバーに入力して開きます。

## 🎮 機能

- **デモモード**: API Keyなしですぐに試せます。
- **音声会話**: マイクで話しかけるとAIが応答します。
- **音声読み上げ**: AIの応答が英語で読み上げられます。
- **リスニング練習**: AIのテキストを隠して音声だけに集中できます。
