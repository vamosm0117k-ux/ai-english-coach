# アプリの公開（デプロイ）手順

このアプリをインターネット上で誰でも使えるようにするための手順です。
無料で使える「Vercel（バーセル）」というサービスを利用します。

## 事前準備: GitHubへのアップロード
まず、このプロジェクトのコードをGitHubにアップロードする必要があります。
（※すでにGitHubにアップロードされている前提で進めます）

## Vercelでの公開手順

1. **Vercelのアカウント作成**:
   - [Vercel公式サイト](https://vercel.com/) にアクセスし、"Sign Up" からGitHubアカウントで登録・ログインします。

2. **新しいプロジェクトを作成**:
   - ダッシュボードの "Add New..." ボタンから "Project" を選択します。
   - "Import Git Repository" のリストから、このアプリのリポジトリ（`ai-english-coach` など）の "Import" ボタンをクリックします。

3. **環境変数の設定 (重要)**:
   - "Configure Project" 画面が表示されます。
   - **"Environment Variables"** という項目をクリックして展開します。
   - ここにGemini API Keyを設定します（これで他の人がキーを入力せずに使えるようになります）。
     - **Key**: `VITE_GEMINI_API_KEY`
     - **Value**: `あなたのGemini APIキー` （`AIza`から始まる文字列）
   - "Add" ボタンをクリックして追加します。

4. **デプロイ実行**:
   - "Deploy" ボタンをクリックします。
   - 1〜2分待つと、ビルドが完了し、完了画面が表示されます。

5. **公開完了**:
   - 表示された画像をクリックするか、"Visit" ボタンを押すと、公開されたアプリが開きます。
   - URL（例: `https://ai-english-coach-project.vercel.app/`）を友だちやSNSで共有すれば、誰でもあなたのアプリを体験できます！

## 注意点
- **API利用料**: 公開されたアプリでAPIが使われると、あなたのGoogle Cloudアカウントの課金枠が消費されます。心配な場合は、Google Cloud Consoleで「予算アラート」や「使用量制限（Quota）」を設定することをお勧めします。
